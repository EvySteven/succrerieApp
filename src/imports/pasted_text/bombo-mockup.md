Crée une maquette complète pour "Bombo", une boutique de friandises 
artisanales en ligne. Le site est moderne, coloré, chaleureux et 
gourmand. Il cible tous les âges, avec une identité visuelle douce 
et joyeuse.

Palette de couleurs :
- Primaire    : Rose bonbon   #FF6B9D
- Secondaire  : Jaune sucre   #FFD93D
- Accent      : Violet candy  #A855F7
- Fond clair  : Crème doux    #FFF8F0
- Texte       : Brun chocolat #3D1C02
- Succès      : Vert menthe   #4ADE80
- Fond carte  : Blanc pur     #FFFFFF

Typographie :
- Titres   : "Nunito" ou "Baloo 2" — gras, arrondi, joyeux
- Corps    : "Poppins" Regular 16px — lisible et moderne
- Accents  : "Pacifico" — pour le logo et les grandes affiches

Style général :
- Coins arrondis partout (border-radius : 16–24px)
- Ombres douces (box-shadow légère, pas agressive)
- Illustrations de bonbons / confettis en arrière-plan (subtils)
- Boutons en dégradé rose → violet
- Icônes rondes et ludiques
- Beaucoup d'espace blanc pour respirer

État ouvert (fenêtre chat 380x500px) :
- Header rose : avatar robot Bombo 🤖 + "Bombo Assistant" 
  + statut "En ligne ●" + bouton ✕ pour fermer
- Zone messages avec bulles : 
    → Bulles IA : fond blanc, texte sombre, alignées à gauche
    → Bulles client : fond rose, texte blanc, alignées à droite

  Comment puis-je t'aider ?"

  "Suivre ma commande" | "Voir les promos" | "Parler à un humain"
- Barre de saisie en bas : champ texte + bouton envoi rose
- Si client clique "Parler à un humain" → 
  message "Un gérant va te rejoindre dans quelques instants 👋"
  et transition vers chat humain (indicateur "Gérant en ligne")

État réduit (après utilisation) :
- Petite bulle discrète avec icône 💬 et tooltip "Besoin d'aide ?"
